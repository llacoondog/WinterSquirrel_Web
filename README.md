광고에서 많이 나오던 그 게임을 모티브로 만들었습니다.

모닥불에서 열기를 공급받으며 나무 근처에서 나무를 캐고 모닥불에 연료도 넣고 업그레이드도 하면 됩니다.

게임의 최종 목적은 맵 사방에 흩어진 도토리 4개 모으기입니다.

단, 보유 열기와 상관 없이 모닥불에서 10초 이상 떨어지면 한기가 가속되니 조심하세요!
